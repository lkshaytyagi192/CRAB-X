## 📖 Overview

**CRAB-X** is a 3D-printed robotic crab designed to explore bio-inspired locomotion, embedded control systems, and autonomous robotics.

The project combines mechanical design, servo-based actuation, embedded programming, and future AI capabilities into a modular platform that can be easily upgraded with sensors, cameras, and intelligent navigation algorithms.

Whether you're learning robotics or developing advanced walking robots, CRAB-X provides an accessible and expandable platform for experimentation.

- 🦀 Bio-inspired crab walking mechanism
